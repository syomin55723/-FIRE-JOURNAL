# FIRE Stock Growth Journal (FIRE 股票成長日誌)

這是一個專為追求 **FIRE (Financial Independence, Retire Early)** 目標的投資者設計的高級股票成長日誌應用程式。採用現代化的 **Frosted Glass (磨砂玻璃)** 視覺風格，提供沈浸式且直觀的投資管理體驗。

## ✨ 核心設計特點

- 💎 **磨砂玻璃視覺 (Frosted Glass UI)**：利用現代 CSS 背景模糊與半透明層次，營造高級機構級軟體質感。
- 🌈 **動態網格背景 (Mesh Gradients)**：隨視角變化的動態流動色彩背景，提升數據讀取的舒適度。
- 📊 **專業數據視覺化**：整合 Recharts 提供流暢的交互式圖表，追蹤資產變化與產業權重。
- 📱 **延展性佈局**：完全響應式設計，適配桌面端與移動裝置單手操作。

## 🚀 主要功能模組

1. **資產看板 (Dashboard)**：即時追蹤投資組合總值、獲利因子與近期交易動態。
2. **投資組合 (Portfolio)**：詳細的持倉清單與產業權重分佈分析。
3. **交易日誌 (Journal)**：記錄每筆交易的心理狀態、技術面信號與執行細節。
4. **觀察名單 (Watchlist)**：監控潛在 Alpha 機會，設定價格提醒與信號偵測。
5. **深度分析 (Analysis)**：提供風險回報比、勝率分析及防範認知偏誤的數據洞察。
6. **個股詳情 (Stock Detail)**：專業的 K 線模擬、技術分析筆記與策略契合度檢查清單。

## 🛠️ 技術棧 (Tech Stack)

- **Frontend**: React 18 (TypeScript)
- **Styling**: Tailwind CSS 4.0
- **Animation**: motion/react (Framer Motion)
- **Icons**: Lucide React
- **Charts**: Recharts & D3.js
- **Build Tool**: Vite

## 📦 安裝與運行

```bash
# 克隆專案
git clone https://github.com/syomin55723/-FIRE-JOURNAL.git

# 進入目錄
cd -FIRE-JOURNAL

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```
